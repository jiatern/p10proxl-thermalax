Inspired by [Thermalax](https://xdaforums.com/t/mod-thermalax-relaxed-thermal-throttling.4686085/).

This mod increases temperature threshold values by around 3-5C.

The module itself is not provided as I cannot guarantee it works for your devices, as google changes these files from time to time. 

The 2 json files should be replaced in the `/system/vendor/etc/` directory using a magisk/kernelSU module.
